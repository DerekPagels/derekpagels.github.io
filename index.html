Entrada de blog de ejemplo

Viernes, 20 de Enero, 2023 | Derek Pagels

Ésta entrada sirve de ejemplo para una entrada personalizada en el blog personal del autor. Los componentes principales para éste son el título, la fecha, autor y cuerpo.

Las entradas de blog pueden tener varios formatos, como la regla horizontal dividendo éste párrafo y el anterior. A continuación se mostrarán diferentes ejemplos de formato para las entradas de blog.

Citas de bloques.

Se pueden citar diferentes autores y/o usar el formato a la preferencia del autor. Un ejemplo de éste elemento sería:

"Aquí iría una cita, comentario y/o bloque de texto que el autor quiera enfatizar"
Derek Pagels

Se pueden usar otros tipos de texto dentro del bloque de texto. En este caso, por ejemplo, se utilizó un span con clase text-italic para distinguir entre la cita y su autor.

Listas

Se pueden usar listas para ordenar diferentes ideas y conceptos. Por ejemplo, se presenta a continuación una lista sin numeración:

  • Tenemos una lista sin numeración, también conocida como lista desordenada.
  • Tiene tres elementos.
  • Los elementos son distinguidos con puntos de viñeta.

A continuación, un ejemplo de una lista ordenada:

  1. Tenemos una lista numerada, también conocida como lista ordenada.
  2. Cada elemento es distinguido con un número en orden ascendente.

HyperText Markup Language (HTML)
El lenguaje usado por un navegador web para mostrar una página.
Cascading Style Sheets (CSS)
Lenguaje usado por un navegador web para dar estilo a una página HTML.
JavaScript (JS)
Lenguaje de programación usado por un navegador web para ejecutar contenido y cambiar páginas dinámicamente.

Inline HTML elements

HTML defines a long list of available inline tags, a complete list of which can be found on the Mozilla Developer Network.

  • To bold text, use <strong>.
  • To italicize text, use <em>.
  • Abbreviations, like HTML should use <abbr>, with an optional title attribute for the full phrase.
  • Citations, like — Mark Otto, should use <cite>.
  • Deleted text should use <del> and inserted text should use <ins>.
  • Superscript text uses <sup> and subscript text uses <sub>.

Most of these elements are styled by browsers with few modifications on our part.
